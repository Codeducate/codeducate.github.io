Our Mission

We are a team of students from Thomas Jefferson High School for Science and Technology determined to foster computer science among middle school students.

Hands-On Learning

Students will constantly be in contact with the language with hands-on tutorials and lessons, not just lectures.

Guest Speakers

We'll have guest speakers throughout the year from local tech companies. They'll talk about the cool things they've done!

Projects

Students will work on a year-long project using the skills they've learned and present it at the end of the year.

Our Courses

We teach a wide range of courses, ranging from web and iOS development to competitive programming.

Web Development Fundamentals: HTML and CSS

Learn how to create and design your own website using basic languages such as HTML and CSS!

Advanced Web Development: JavaScript

Make your website more professional with JavaScript, the programming language of the Web!

Fundamentals of Programming in Python

Learn how to program in Python, a powerful language used by sites such as YouTube and Dropbox!

Fundamentals of Programming in Java

Learn how to program in Java, a fundamental language used by programmers around the world!

Competitive Programming

Learn fundamental programming techniques to compete in national programming competitions!

Programming for iOS

Learn how to create and design iOS apps for Apple using Xcode and Swift.
