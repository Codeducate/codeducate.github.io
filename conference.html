Activities

Beginner: These activities require no prior programming experience and can be done by anyone!
  • HTML 101: Learn the basics of HTML!
  • All About Me: Create an HTML webpage all about you!
  • Ships Ahoy: Create an HTML webpage to advertise for your pirate crew!
Intermediate: Although these activities require no prior programming experience, they’ll be a challenge! It’s recommended that you’ve went to at least “HTML 101” before attending these activities.
  • CSS 101: Learn the basics of CSS!
  • Prettify That: Use HTML and CSS to create a webpage for your design studio!
Advanced: These activities are definitely a challenge and are for those who have prior programming experience or for those who are looking to challenge themselves. It’s recommended that you’ve went to at least “HTML 101” and “CSS 101” before attending these activities.
  • JavaScript 101: Learn the basics of JavaScript!
  • The Ultimate Test: Create a quiz to test your friends and parents!
  • Intro to Debugging: Learn how to debug your code!
